# v1.1.1

* **New intent type:** `fw_rule` (Firewall Rule) for declarative firewall policies.
* **New vendor support:** Firewall controller adapter and primitive rendering for centralized firewall platforms (Palo Alto Panorama, Fortinet FortiManager, generic REST-based firewall manager).
* **Jinja2 templates:** `fw_rule` templates added for all supported device platforms (Cisco IOS-XE/IOS-XR/NX-OS, Juniper Junos, Aruba AOS-CX, Arista EOS).
* **Updated intent documentation:** Security domain now includes firewall rule intents, and the intent type count is updated to 134.
